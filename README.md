# maskMap

시중에 나와 있는 공적마스크 판매처 제공 웹은 정렬기준이 없어 <strong>거리 순, 입고 순 정렬 기능</strong>을 추가했습니다.



<h3>-Stack <br>

  framework : Spring Boot
  
  Docker-load balancing(4 jdk containers) with Nginx(reverse proxy)
  
  CI : travis CI 
  
 storage : AWS S3
  
  DB: Mysql
  
  deploy : AWS CodeDeploy
